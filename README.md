# my_website

**Install Node.js**

Using the following link [https://nodejs.org/en/download](https://nodejs.org/en/download) to install Node.js. 

To install Node.js via binary archive on Linux use the link [https://github.com/nodejs/help/wiki/Installation#how-to-install-nodejs-via-binary-archive-on-linux](https://github.com/nodejs/help/wiki/Installation#how-to-install-nodejs-via-binary-archive-on-linux)

To confirm that Node has been successfully installed, run run the command `node -v`. This will display the current version of Node installed on your computer.

**Install React**
 
To install the create-react-app package globally run run the command `npm I -g create-react-app`. Note that Ttis is one time installtion for the device

To create react app run `create-react-app app_name`. Replace the app_name with the desired app name.

**Run UI**

To run app in developent mode run the command `npm start`

To deploy project run run the command `npm run build`
